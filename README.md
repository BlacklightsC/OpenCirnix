# OpenCirnix
워크래프트3 외부 프로그램 Cirnix의 오픈소스 버전입니다.

**개발에 사용된 IDE: Visual Studio 2019**

**테스트 완료된 운영체제: Windows 10 2004**

**요구사항**
- Warcraft III 1.28.5.7680 (1.28f)
- .NET Framework 4.6.2 이상의 버전
- [CirnoLib](https://github.com/BlacklightsC/CirnoLib.git)
